Course Title:
Data Structure and Algorithm

Course Code:
ICC 4

Learning Outcome 1 (LO1):
Demonstrate a foundational understanding of data organization and the mathematical analysis of algorithm performance.

Topics under LO1:
1. Introduction to Algorithms and Complexity
   - Characteristics of an Algorithm
   - Big O, Omega, and Theta Notations
   - Best, Average, and Worst-case Scenarios

2. Abstract Data Types and Arrays
   - Logical vs Physical Data Structures
   - Memory Allocation (Static vs Dynamic)
   - Multi-dimensional Array Mapping

3. Foundations of Data Organization
   - Data Representation Concepts
   - Efficiency in Data Handling
   - Role of Data Structures in Computing


Learning Outcome 2 (LO2):
Implement and manipulate linear and non-linear data structures to manage information flow efficiently.

Topics under LO2:
1. Linked List Architectures
   - Singly, Doubly, and Circular Linked Lists
   - Dynamic Memory and Pointer Manipulation
   - Insertion and Deletion Algorithms

2. Stacks and Queues
   - LIFO and FIFO Principles
   - Implementation using Arrays and Linked Lists
   - Applications (Undo, Buffers)

3. Tree Structures
   - Binary Trees and Binary Search Trees
   - Tree Traversal Methods
   - Balanced Trees and AVL Introduction


Learning Outcome 3 (LO3):
Utilize advanced algorithmic strategies including recursion, sorting, and searching to optimize computational tasks.

Topics under LO3:
1. Searching and Hashing Techniques
   - Linear vs Binary Search
   - Hash Functions and Collision Resolution
   - Open Addressing vs Chaining

2. Sorting Algorithms
   - Bubble, Selection, and Insertion Sort
   - Merge Sort and Quick Sort
   - Counting and Radix Sort

3. Recursive Problem Solving
   - Base Case and Recursive Step
   - Recursion vs Iteration
   - Classic Problems (Fibonacci, Hanoi)


Learning Outcome 4 (LO4):
Construct and evaluate complex data models like graphs and hash tables to solve real-world engineering problems.

Topics under LO4:
1. Graph Theory and Representation
   - Graph Structures and Components
   - Adjacency Matrix and List
   - Real-world Graph Applications

2. Graph Traversal and Pathfinding
   - Breadth-First Search (BFS)
   - Depth-First Search (DFS)
   - Dijkstra’s Algorithm

3. Algorithm Optimization and Selection
   - Space-Time Trade-offs
   - Data Structure Selection Strategies
   - Performance Optimization Techniques
